# Investment Analysis Projects

This repository contains a collection of personal investment and macroeconomic analysis projects implemented in Python and Jupyter notebooks.

## Projects

### Portfolio Performance & Risk Analysis
Time-weighted performance analysis of a personal equity portfolio, including risk metrics, drawdowns, and benchmark comparison.

➡️ [`portfolio-performance/`](portfolio-performance)



### Mortgage Rate & Housing Market Analysis
Statistical analysis and visualization of U.S. mortgage rates and related macro indicators.

➡️ [`mortgage-rate-analysis/`](mortgage-rate-analysis)

## Tools & Methods
- Python, pandas, NumPy, matplotlib
- Jupyter notebooks
- Financial performance metrics and factor models
