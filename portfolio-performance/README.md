# Portfolio Performance & Risk Analysis

This project evaluates the performance of a personal equity portfolio using monthly account-level data.

## Key Features
- Time-weighted return calculation net of cash flows
- Risk metrics including Sharpe, Sortino, and max drawdown
- Visual analysis of cumulative returns and drawdowns

## Data
- Monthly account balance and activity data exported from Fidelity
